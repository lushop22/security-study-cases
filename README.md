# Casos de Estudio — Ciberseguridad

Colección de análisis forenses educativos sobre incidentes de ciberseguridad relevantes. Las páginas están diseñadas para ser usadas como material de apoyo en cursos de redes y seguridad informática.

> **Aviso importante:** Este repositorio es de carácter **estrictamente educativo**. Todo el contenido está basado en fuentes públicas, reportes técnicos de organizaciones de seguridad y artículos periodísticos de investigación. Ninguna información fue obtenida de forma ilegal ni representa conocimiento propio o propietario. Las fuentes están citadas en cada caso.
